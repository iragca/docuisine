# Branding

This page discusses Docuisine's branding and theming.

## Theme

The colors listed here are the significant choices we have made to represent Docuisine. Colors not mentioned are left to defaults and standard colors offered by [TailwindCSS](https://tailwindcss.com/) and [shadcn](https://ui.shadcn.com/).

| Domain     | Variant              | Hex       | oklch              | TailwindCSS |
| ---------- | -------------------- | --------- | ------------------ | ----------- |
| Logo       | Primary              | `#795649` |                    |             |
| Logo       | Secondary            | `#EDEAD5` |                    |             |
| UI - Light | Primary              |           | 27.9% 0.077 45.635 | `amber-950` |
| UI - Light | Primary-Foreground   |           | 0.985 0 0          |             |
| UI - Light | Secondary            |           | 98.7% 0.022 95.277 | `amber-50`  |
| UI - Light | Secondary-Foreground |           | 27.9% 0.077 45.635 | `amber-950` |
| UI - Light | Accent               |           | 96.2% 0.059 95.617 | `amber-100` |
| UI - Light | Accent-Foreground    |           | 27.9% 0.077 45.635 | `amber-950` |

These colors are implemented as code in the [frontend repository's `index.css`](https://github.com/iragca/docuisine-react/blob/master/src/index.css).

## Icons and Other Assets

The logo and the favicon can be edited using its [`.psd` file](../../assets/branding/docuisine.psd). The typography has its own [`.psd` file](../../assets/branding/docuisine-typography.psd). You can edit for free using [Photopea](https://www.photopea.com/).

The assets on display here can be downloaded as is by right clicking on the selected image and save it.
